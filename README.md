# Portfolio

Personal portfolio of Phanidhar Akula, a researcher, engineer and builder working at the seam of urban simulation, HPC and AI.

Live site: [phanidhar.dev](https://phanidhar.dev)

## About

This is a single page portfolio that brings together my research, projects, experience and writing in one place. It covers my M.S. thesis work on reproducible traffic simulation, my open source contributions to digital twin and population synthesis frameworks, the AI study platform I build and operate, and a set of peer reviewed publications and earlier engineering work.

## Highlights

- SimForge, a reproducible cross simulator benchmarking framework for urban traffic
- Cityscape, an open source population synthesis and geospatial demand framework
- Lumi AI, a production AI study platform live with real student users
- Peer reviewed publications in deep learning and topic modeling

## Tech Stack

- React with TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Three.js with React Three Fiber

## Deployment

The site is deployed on Vercel. Pushes to the release branch ship to production automatically.
